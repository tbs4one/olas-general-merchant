# Olas General Merchant & Store Items

A modern, dynamic website for **Olas General Merchant & Store Items** — a premium grocery and drinks store in Ajegunle, Lagos.

## Features

- **3D animated logo** — Interactive Three.js logo with rotating ring, floating product cubes, and mouse tracking
- **Product catalog** — Rice, flour (Dangote, Honeywell, Bua, Golden Penny), drinks, and customised gift items
- **Drinks advert gallery** — Carousel showcasing Coca-Cola, Fanta, Pepsi, Big Cola, Sprite, yoghurt, spirit, Monster, Red Bull & more
- **Order system** — Order form with email submission and WhatsApp integration
- **Responsive design** — Works on desktop, tablet, and mobile

## Quick Start

Open `index.html` directly in your browser, or run a local server:

```bash
# Python
python -m http.server 8080

# Node.js (if npx is available)
npx serve .
```

Then visit: **http://localhost:8080**
